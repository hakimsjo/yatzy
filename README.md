# Yatzy spelet

Ett litet hjälpmedel för att spela yatzy utan tärningar. Slå ditt slag och välj vilka tärningar du vill spara. Lägg in poängen i tabellen. Alla kan ha den på sin mobil eller så kan man samsas om en.
